Map Explorer: See Around

"Map Explorer: See Around" is an interactive exploration game designed using Unreal Engine 5. Players take on the role of a cartographer navigating through beautifully crafted environments, uncovering hidden landmarks, and mapping uncharted territories. The game features dynamic weather changes, realistic landscapes, and engaging quests that encourage players to explore every corner of the map.

Technologies Used:

  • Unreal Engine 5: For creating immersive 3D environments and real-time rendering.
  • Blueprints: For visual scripting, allowing for interactive gameplay mechanics without extensive coding.
  • C++: To implement advanced functionalities and optimize performance.
  • Quixel Megascans: For high-fidelity assets and textures, enhancing the visual richness of the game world.
  • Audio Design Tools: For creating ambient sounds and music that enrich the exploration experience.

RemixHub

RemixHub is a dynamic music app developed using Flutter that enables users to discover, stream, and curate their favorite tracks effortlessly. The app offers an engaging user experience with an intuitive interface, allowing users to create custom playlists, explore new music based on their tastes, and enjoy high-quality audio streaming. With personalized recommendations and a vast music library, RemixHub brings a fresh approach to music enjoyment.

Technologies Used:

  • Flutter: For building a cross-platform mobile application, ensuring smooth performance on Android devices.
  • Dart: As the programming language for implementing app features and logic.
  • FastAPI: To create a robust backend API that handles user requests, music data management, and real-time interactions.
  • Firebase: For user authentication and real-time database functionalities.
